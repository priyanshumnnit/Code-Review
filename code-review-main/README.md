An intelligent, modern, and fully interactive AI-powered code review system that analyzes your code using Google Gemini (Memini) and provides detailed feedback, improvements, and error detection.

This project includes a beautiful full-screen UI with a live code editor and real-time AI review panel — perfect for developers, students, and anyone who wants to improve their code quality instantly.

Features
AI-Based Code Review

Uses Google Gemini (Memini) to analyze code.

Provides detailed reviews, suggestions, refactoring tips, and bug analysis.

Supports multi-language code inputs (C++, JavaScript, Python, etc).

 Interactive Code Editor

Built with React Simple Code Editor + PrismJS.

Real-time syntax highlighting.

Smooth typing and fullscreen interface.

 Beautiful Modern UI

Fullscreen split layout (Code → Review).

Glassmorphism design.

Auto-wrap markdown review output.

Dark modern theme.

 Fast & Lightweight Backend

Node.js + Express server.

Clean architecture (controllers + services).

Handles communication with Google Gemini API.

 Tech Stack
Frontend

React.js

PrismJS

React-Markdown + Highlight.js

Axios

CSS with modern styling

Backend
